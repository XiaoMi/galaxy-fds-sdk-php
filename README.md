文件存储(FDS)PHP SDK使用介绍
=============================
##### 安装第三方依赖

    ./composer.phar install

##### 运行测试程序，如果测试程序运行成功，则说明环境配置正确:

    php ./examples/galaxy-fds.php (注意: 需要在样例中设置正确的App Key和Secret)
